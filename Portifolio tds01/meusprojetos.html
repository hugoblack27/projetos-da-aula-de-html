Padaria do joão

Na aula de Análise de Desenvolvimento de Sistemas do Senac Largo Treze, realizamos uma atividade onde usamos o SQL MariaDB para criar o banco de dados e o Sublime Text para planejar o código. O objetivo foi estruturar o banco, definir tabelas e realizar operações básicas de CRUD, enquanto organizávamos as ideias do sistema no Sublime. A atividade ajudou a fortalecer nossos conhecimentos sobre integração de banco de dados e desenvolvimento de sistemas.
